# 🎲 Real-Time Bingo Game

A modern, real-time multiplayer Bingo game built with **React**, **Node.js**, and **Socket.io**.  
Play with friends instantly by creating or joining a room.

---

## 🚀 Features

- 🎮 Create or join game rooms
- 👥 Real-time multiplayer (Socket.io)
- 🎯 Interactive Bingo card
- 🔢 Live number calling system
- 🏆 Winner detection (BINGO)
- 📱 Fully responsive UI (mobile + desktop)

---

## 🛠️ Tech Stack

### Frontend
- React
- Tailwind CSS
- React Router

### Backend
- Node.js
- Express
- Socket.io
